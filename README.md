# Online Food Ordering System (OFOS)

## Overview
The Online Food Ordering System (OFOS) is a prototype web application built using Appsmith. It demonstrates a simple food ordering workflow where customers can browse restaurants, view menus, and place orders. This prototype is based on the OFOS Software Requirements Specification (SRS) and is intended for academic/demo purposes.

## Features
- View list of restaurants with cuisine, rating, and delivery time
- Navigate from Restaurant List page to Menu & Order page
- View menu items for the selected restaurant
- Capture customer name and address
- Select one or more menu items to order
- Store latest order in Appsmith store and show confirmation message
